# __Sampling（抽樣）__
> 初學統計基本概念

- 簡易隨機抽樣(Simple random Sampling)
```
每個可能樣本被抽出的機率相同
```
- 分層隨機抽樣(Stratified random Sampling)
```
母體所有元素先被分類成群，稱為「層」

· 層內元素越像，層間差異大 -> 增加樣本代表性
```
- 群集抽樣(Cluster Sampling)
```
先將母體分成多個群，再隨機抽出一群，對其進行調查

· merit : 有效降低抽樣成本
```
- 系統抽樣(Systematic Sampling)
```
在母體很大的情況，抽取的樣本數多時，比簡易隨機抽樣還簡單
```
## 非機率抽樣
- 便利抽樣(Convenience Sampling)
```
樣本是否被抽出根據其便利性來判定，無法知道被抽中的機率
```
- 判斷抽樣(Judgment Sampling)
```
研究者必須了解研究對象，選出能代表母體的樣本，此樣本的參考性依研究者判斷而定

＊ E.g. -> 一名記者選出他認為最能反映議員觀點的議員來採訪
```
